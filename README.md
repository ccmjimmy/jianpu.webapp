簡譜.webapp
=========

簡譜網路應用程式 Jianpu.webapp - a numbered musical notation web application


本計畫目前處與起始狀態，更多的訊息，請到本專案的維基

This project is in the initial state, please go to the wiki for further information.

https://github.com/jianpu/jianpu.webapp/wiki


## 更新 Update (2015-9-8)

- 本計畫的 [123 語言](https://github.com/jianpu/jianpu.webapp/wiki/123%E8%AA%9E%E8%A8%80) 部分，已移至  [musje-123-notation][] 專案繼續發展。由於英文參考資料較多與打字的方便，皆使用英語。 (Please move to [musje-123-notation][] for specifications of the *musje 123 notation*)

- 解析器 (parser)、資料模型 (data model)、排版 (layout) 與渲染 (renderer) 引擎、及播放 (player) 則在 [musje](https://github.com/jianpu/musje) 專案上開發。

[musje-123-notation]: https://github.com/jianpu/musje-123-notation

## 本專案的未來發展 Future of this repo (2015-9-8)

本專案停止開發。

未來可能的計劃，例如上更新，將不會在本專案裡面進行開發。本專案不會繼續發展，僅提供舊有的資料，與未來發展相關的連結。歡迎有志之士接管本專案。

The repo is closed. Future project may, as the above update, be developed elsewhere. People who interested in this repo are more than welcome to take over this repo.
